## 📌 문제: AI 데이터 파이프라인 최적화
당신은 MLOps 인프라 엔지니어입니다. 새로운 대형 언어 모델(LLM)을 학습시키기 위해 총 $N$개의 데이터 전처리 및 학습 작업(Job)을 파이프라인으로 구성했습니다.

각 작업은 1번부터 $N$번까지 고유 번호가 부여되어 있으며, 작업마다 소요되는 시간이 다릅니다. 또한, 어떤 작업은 특정 선행 작업이 완전히 끝나야만 시작할 수 있는 의존성(Dependency)을 가집니다. 선행 작업이 없는 작업들은 서버 자원이 허락하는 한 동시에(병렬로) 실행될 수 있습니다.

작업의 개수 N, 각 작업의 소요 시간 배열 times, 작업 간의 선행 조건이 담긴 배열 dependencies가 주어질 때, 모든 작업이 완료되는 데 필요한 최소 시간을 반환하는 함수를 완성하세요.

## 제한사항
- N은 1 이상 1,000 이하의 자연수입니다.
- times의 길이는 N이며, times[i]는 i+1번 작업의 소요 시간(분)입니다. (1 이상 1,000 이하)
- dependencies의 각 원소는 [A, B] 형태이며, 이는 A 작업이 완전히 완료되어야 B 작업을 시작할 수 있음을 의미합니다.
- 순환 참조(A $\rightarrow$ B $\rightarrow$ C $\rightarrow$ A)가 발생하는 등 작업을 완료할 수 없는 예외 케이스는 주어지지 않습니다.


| N | times | dependencies | result |
|---|---|---|---|
| `4` | `[10, 20, 10, 20]` | `[[1, 2], [1, 3], [2, 4], [3, 4]]` | `50` |
| `3` | `[5, 5, 5]` | `[]` | `5` |
| `5` | `[10, 10, 20, 20, 30]` | `[[1, 3], [2, 3], [3, 4], [4, 5]]` | `80` |

### 예시 1번 설명:
- 1번 작업이 끝나는 시간: 10분
- 2번, 3번 작업은 1번이 끝난 직후(10분) 병렬로 시작.
  - 2번 완료 시간: 10분 + 20분 = 30분
  - 3번 완료 시간: 10분 + 10분 = 20분
- 4번 작업은 2번과 3번이 모두 끝나야 시작 가능. 즉, 늦게 끝나는 2번 기준인 30분에 시작 가능.
  - 4번 완료 시간: 30분 + 20분 = 50분
- 전체 최소 완료 시간은 50이 됩니다.
